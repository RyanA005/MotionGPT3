# How to Install Dependancies

## Step 1 - Environment
### Prerequisites 
- Python 3.10.x
- Visual Studio Build Tools

### Python Environment
Windows
```bat
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
```

Linux
```bash
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
```
